# sni-knowledge

Shared knowledge repository for the SNI strategy team. Git-based store of curated
meeting summaries, strategic threads, decisions, and references. Content repo, not code.

---

## Key Facts

- Type: Content repository (markdown + YAML frontmatter), not a code project
- Pipeline: private vault → data-layer ingestion → human review → promotion here
- Structure: `knowledge/` (meetings, threads, decisions, references), `templates/`, `taxonomy/`
- Status: Low-activity (2 commits, last March 2026). Content pipeline not yet operational.

## Guardrails

- Follow templates in `templates/` for new artifacts
- Follow naming conventions in `docs/naming-conventions.md`
- No binary files (PPT, Excel, PDF)
- This repo is an output of a promotion pipeline — direct edits are secondary to the pipeline flow
