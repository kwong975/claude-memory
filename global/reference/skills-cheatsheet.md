# Claude Code Skills Cheat Sheet

## Re-entry
/resume — assemble repo + project briefing (git state, next items, known issues)

## Design
/design <topic> — structured thinking before implementation (problem → options → recommendation)

## Session Close
/wrapup — summarize work, propose project updates, propose explicit memory updates (approval required)

## Memory Hygiene
/reflect review — scan memory for stale/duplicate/contradictory entries
/reflect clean — apply approved cleanup

## System Health
/audit — check repo coverage, memory consistency, budgets, hooks
/audit <area> — scoped check (repos, memory, budgets, hooks, skills)

---

## Mental Model

- Start → /resume
- Think → /design
- End → /wrapup
- Clean → /reflect
- Check → /audit
